Portfolio

MakeTogether

A social media platform for web designers and web developers to meet each other, post projects they are seeking collaborators on, and have conversations. Ruby on Rails web app utilizing jQuery and AJAX to enhance user experience. GitHub OAuth and a custom Dribbble OAuth solution. Bootstrap framework used to enhance design. Deployed live on Heroku.

2048 Clone

A web application where users press arrow keys to move tiles and try to get to the 2048 tile. Tiles merge when two of the same number touch. Written in vanilla javascript. Deployed live using GitHub Pages.

The Muse Job Search

A Ruby on Rails web application using The Muse's public API that allows users to search for jobs based on company, job level, job category, and/or location. jQuery and AJAX used to enhance user experience and Bootstrap framework for design and RSpec for testing. Deployed live on Heroku.

LES Film Festival

A web application for a film festival where users, judges, and admins can sign up, vote, leave reviews, and comments. Ruby on Rails back-end, utilizing jQuery and AJAX to enhance user experience. Bootstrap for design and RSpec for testing. Deployed live on Heroku.

Recent Tweets

A Ruby on Rails web application built using Twitter's API that allows a user to search for the 10 most recent tweets of a user by searching their handle. Bootstrap used to enhance design. Deployed live on Heroku.

Travel Overflow

A web application, built in Ruby and Sinatra using polymorphic associations, where users can ask travel related questions, answer questions, and vote and comment on questions, answers, and other comments. jQuery and AJAX used to enhance user experience. Built using RESTful conventions and custom CSS for design.
