Portfolio

MakeTogether

A social media platform for web designers and web developers to meet each other, post projects they are seeking collaborators on, and have conversations. Ruby on Rails web app utilizing jQuery and AJAX to enhance user experience. GitHub OAuth and a custom Dribbble OAuth solution. Bootstrap framework used to enhance design. Deployed live on Heroku.

2048 Clone

A web application where users press arrow keys to move tiles and try to get to the 2048 tile. Tiles merge when two of the same number touch. Written in vanilla javascript.

LES Film Festival

A web application for a film festival where users, judges, and admins can sign up, vote, leave reviews, and comments. Ruby on Rails back-end, utilizing jQuery and AJAX to enhance user experience. Bootstrap for design and RSpec for testing.
