My Experience Building a Website

Building a website has been really fun and very fulfilling. It is awesome to see my HTML and CSS come alive on the page. I've learned a lot about HTML 5 semantics and many CSS topics, such as positioning. The biggest challenge I've had was desigining my site and getting elements to fall where I want them to on the page. I look forward to learning about how to format my site to look great on all browsers as well as mobile devices. Phase 1 starts on Monday and I cannot wait to learn more and get closer to becoming a developer!
