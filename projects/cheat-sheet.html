A Quick Guide to SQL

  • SELECT

    used to select data from a database

    SELECT column_name,column_name
    FROM table_name;

    AND

    SELECT * FROM table_name;

    Using* selects ALL content from table

  • WHERE

    used to extract only data that meets a criterion

    SELECT column_name,column_name
    FROM table_name;
    WHERE column_name operator value;

    Operators:

    >, <, =, >=, <=, <> or !=, BETWEEN, LIKE, IN

  • AND & OR

    AND/OR extracts only data that fits

    the 1st AND/OR 2nd condition

    SELECT column_name,column_name
    FROM table_name;
    WHERE column_name operator value;
    AND/OR column_name operator value;

  • ORDER BY

    used to sort the result-set by 1 or more columns

    SELECT column_name,column_name
    FROM table_name;
    ORDER BY column_name ASC|DESC,
    column_name ASC|DESC;

    ASC/DESC=

    ascending/descending order

  • LIKE

    used in WHERE clause to search for
    specified pattern in a column

    SELECT column_name,column_name
    FROM table_name;
    WHERE column_name LIKE pattern;

    Use % before/after keyword in order to
    find results containing search word.
    (% after for starts with, before for begins with)

  • IN

    used in WHERE clause to search for
    multiple values in a column

    SELECT column_name,column_name
    FROM table_name;
    WHERE column_name IN (value1,value2,...);

    The same syntax as LIKE is used

  • BETWEEN

    used to select values within a range

    SELECT column_name,column_name
    FROM table_name;
    WHERE column_name BETWEEN value1 AND value2;

    The values can be numbers, text, or dates.

  • UPDATE

    used to update records in a table

    UPDATE table_name
    SET column1=value1,column2=value2,...
    WHERE some_column=some_value;

  • JOIN

    selects all rows from both tables
    as long as there is a match between
    the columns in both tables

    SELECT column_name(s)
    FROM table1
    JOIN table2
    ON table1.column_name=table2.column_name;
